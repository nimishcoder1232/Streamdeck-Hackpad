# Streamdeck-Hackpad
For the Hackclub, Blueprint event, I made a hackpad, that COULD in the future work as a stream deck. While right now this board, has a basic program that just reads the switches, and then displays an output on the lcd. In the future, the actual streamdeck code can be made.
